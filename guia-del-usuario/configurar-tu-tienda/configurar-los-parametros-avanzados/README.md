# Configurar los Parámetros Avanzados

Navegar por el menú de parámetros avanzados es como mirar bajo el capó - aquí es donde todo lo técnico tiene lugar, y donde encontrarás parámetros que tendrás que revisar para asegurarte de que tu tienda funciona sin problemas. Comprobar la información de configuración, configurar direcciones de correo electrónico, hacer copias de seguridad de la base de datos de tu tienda: Estas páginas no se visitan a menudo \(exceptuando la sección correspondiente a la base de datos\), sin embargo, son de cierta importancia, especialmente para el webmaster.
